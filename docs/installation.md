# Installation

Open your terminal or command prompt and run the following commands:

```bash
git clone https://github.com/Adrien-roy/cryptopal_rust.git
cd cryptopal_rust
```
      

Installing Rust with Package Managers

Install Rust using your preferred package manager:

##Example for Ubuntu/Debian

```bash
sudo apt install rustc cargo
```
      

##Example for Fedora

          
```bash
sudo dnf install rust
```
      

##Example for Arch Linux

          
```bash
sudo pacman -S rust
```
      

Building and Testing

Once Rust is installed, build and test your project:

          
```bash
cargo build
cargo test
```
